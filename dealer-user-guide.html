GWM Daily Sales Reporting

Each morning, submit yesterday's completed dealership activity. The report is retrospective only.

Start here

  1. Open the reporting link.
  2. Select your dealership.
  3. Enter your name.
  4. Select Up, Flat or Down.
  5. Confirm the report date.

Enter activity

EnquiryCustomer contact or genuine sales enquiry.
Test drivesCompleted test drives only.
New soldNew customer order written.
Fleet 5+Fleet opportunity of five or more units.
Demo soldDemonstrator vehicle sold.
ForecastCurrent monthly outlook, only update if changed.

Submit

  • Blank fields are treated as zero.
  • Use zero where you want to confirm no activity.
  • Check totals before submitting.
  • Press Submit once and wait for confirmation.

Testing note

During pilot testing, if the same dealer submits the same report date again, the previous submission for that dealer/date is replaced. This keeps testing simple without a dashboard reopen process.
