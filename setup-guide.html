GWM Daily Sales Reporting, simplified setup

Dealer input page to Google Sheets to Power BI. No dashboard layer.

1. What changed

  • The dealer-facing page is now the only live front end.
  • Submissions write straight to the Google Sheet tab named submissions.
  • Power BI reads from the Google Sheet CSV export.
  • The dashboard, reopen workflow, missed-day lock and dashboard setup requirements have been removed for testing.

2. Files to publish

  1. Upload index.html, styles.css and gwm-wordmark.png to GitHub Pages or the chosen static host.
  2. Do not publish dashboard.html. It is not part of this build.
  3. Keep setup-guide.html and dealer-user-guide.html for internal handover or link them only if required.

3. Google Sheet setup

  1. Open the testing Google Sheet.
  2. Create a tab named submissions, or allow the script to create it.
  3. Open Extensions > Apps Script.
  4. Paste the full contents of code.gs.
  5. Replace YOUR_GOOGLE_SHEET_ID_HERE with the actual Google Sheet ID, unless the script is bound to the Sheet.
  6. Save the script project.

4. Reset the sheet for testing

The actual Google Sheet is not reset by the zip file. Reset must be run manually in Apps Script.

  1. If old data needs to be kept, run archiveSubmissionsBeforeReset first.
  2. Then run resetSubmissionsForTesting.
  3. The submissions tab should now contain headers only.

This reset is deliberate and manual so live or historical data is not accidentally deleted.

5. Deploy Apps Script Web App

  1. Go to Deploy > New deployment.
  2. Choose Web app.
  3. Set Execute as to Me.
  4. Set Who has access to Anyone.
  5. Deploy and copy the Web App URL.
  6. Open index.html and replace the value of APPS_SCRIPT_URL with the deployed URL.

6. Power BI connection

Use Power BI Desktop > Get Data > Web with the Google Sheet CSV URL:

https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/gviz/tq?tqx=out:csv&sheet=submissions

Recommended column types:

  • submitted_at: Date/Time
  • report_date: Date
  • is_late: True/False
  • enquiry, test_drives, new_sold, fleet_5_plus, demo_sold, forecast: Whole Number

7. Test process

  1. Open index.html.
  2. Select a dealer, submitted-by name, report date and direction.
  3. Enter a small test quantity for one or two model buckets.
  4. Submit.
  5. Confirm exactly 19 model rows appear in the submissions tab.
  6. Submit the same dealer/date again and confirm the previous 19 rows are replaced.
  7. Refresh Power BI and confirm the rows appear.
