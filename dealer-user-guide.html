GWM Daily Sales Reporting

Each morning, submit yesterday's completed dealership activity. The report is retrospective only.

What to report

Submit yesterday's completed activity only. Do not enter today’s activity and do not enter expected future activity.

  1. Open the reporting link.
  2. Select your dealership, unless your link has already selected it for you.
  3. Enter your name.
  4. Enter the pilot access code only if GWM has supplied one.
  5. Confirm the report date.
  6. Select Up, Flat or Down for daily direction.
  7. Enter completed activity by model bucket.

Metric definitions

FieldEnter this
EnquiryGenuine customer enquiry received for that model. Include walk-in, phone, lead and online enquiry where sales follow-up is required.
Test drivesCompleted test drives only. Do not count booked test drives that have not happened.
New soldNew vehicle customer order or contract written for that model.
Fleet 5+Total units in a fleet opportunity of five or more vehicles.
Demo soldDemonstrator vehicle sold. Do not also count the same unit as new sold.

Forecast

Forecast is not part of this daily activity page. This prevents blank daily fields from wiping monthly forecast reporting. If GWM needs a monthly forecast update, it will be requested separately.

Before pressing submit

  • Check dealer, submitted-by name, report date and direction.
  • Check the live totals at the top of the form.
  • Blank daily activity fields submit as zero.
  • If all totals are zero, only submit if that is a genuine zero-activity day.

After pressing submit

  • Wait for the on-screen receipt before closing the page.
  • The receipt confirms the backend accepted the submission and wrote the rows.
  • Take a screenshot if your Dealer Principal needs proof of submission.

Need to correct a submission?

Submit the corrected full day again for the same dealer and report date. The latest version becomes the active submission. GWM still retains the previous version in the audit log.

Important pilot note

This testing version has no dashboard, no reopen workflow and no missed-day lock. Use the input page only. Power BI handles reporting.
