GWM Daily Sales Reporting

Each morning, submit yesterday's completed dealership activity by model. The report is retrospective only. Dealer submission target is before 10:00am, with a correction window until the internal 12:00pm check.

Core rule

Report completed activity for the selected report date only. Do not enter today's activity, expected future activity or forecast.

DesktopUse the model grid. Enter numbers across the activity columns.
MobileOpen the activity section first, then enter the number beside each model.

Step-by-step

  1. Open the dealer reporting link.
  2. Select your dealership, unless your link has already selected it.
  3. Enter your full name in Submitted by.
  4. Confirm the report date. It should normally be yesterday.
  5. Enter activity by model. A customer may be counted once in each completed stage on the same report date, for example enquiry, test drive and sold, if each stage genuinely occurred.
  6. Check the totals bar and active model count.
  7. Press Submit Daily Report once and wait for the receipt.
  8. Keep the page open until the receipt appears.

Blanks, zeroes and negative numbers

  • Blank fields are treated as zero.
  • Typed 0 is accepted and means zero activity for that model and activity type.
  • Negative numbers are not accepted. If a previous entry was wrong, submit the full corrected day again.
  • If the whole day is zero, the form will ask you to confirm it is a genuine zero-activity day.

Corrections

To correct a mistake, reopen the same reporting link and submit the full corrected day again for the same dealer and report date. The latest version becomes the active reporting version. GWM retains the earlier version in the audit history.

Example: if H6 HEV Sold was entered as 3 but should have been 2, resubmit the full day with H6 HEV Sold as 2. Do not enter -1.

What this page does not collect

  • No forecast.
  • No Fleet 5+ field.
  • No daily direction field.

If GWM needs a forecast update, it will be requested separately.

Troubleshooting

  • No receipt: do not submit again immediately. Check whether the row reached the reporting Sheet or ask GWM to confirm through the status/audit check.
  • Wrong dealer: stop and use the correct dealer link or select the correct dealer before submitting.
  • Access code requested: enter the code supplied by GWM. Leave it blank if no code was supplied.
  • System status says check required: try refreshing once. If it remains, contact the rollout owner before submitting.
