Dealer handover guide

How to use the daily sales report

This form captures completed dealer activity by model. Before 5:00pm, submit yesterday's completed activity. From 5:00pm, submit today's completed activity only when the day is ready to close out.

1. What day to report

Before 5:00pm, submit yesterday's completed activity. From 5:00pm, you can submit today's completed activity if the day is ready to close out. If more activity occurs after submitting, resubmit the full corrected day. Do not enter future activity or expected activity.

2. What each activity means

  • Enquiries: genuine Non-GWM Direct customer contact points, including walk-in, dealer-direct Meta and phone call activity.
  • Test drives: completed test drives.
  • Sold: new vehicle customer orders or contracts written.
  • Demo sold: demonstrator vehicles sold. Do not count the same unit again as Sold.
  • Important: if one customer enquires, test drives and signs on the same report date, record one in each completed stage.

3. How to enter data

On desktop, use the model grid. On mobile, open each activity section first, then enter the number against each model. Blank fields submit as zero.

4. Corrections

To fix a mistake, submit the full corrected day again for the same dealer and date. The latest version becomes active and the earlier version stays in the audit history. Do not submit only the difference.

5. Before submitting

Check dealer, submitted-by name, report date and top totals. If everything is zero, only continue if it was genuinely a zero-activity day.

6. What not to do

  • Do not close the page until the Report received message appears.
Need proof? Take a screenshot of the Report received receipt after submission.
